Stock Market Search Engine

Overview
This application is designed to help you search for information related to the stock market. It extracts data from a predefined web page and provides you with the most relevant information based on your queries.

How to Run
Prerequisites:
Make sure you have Python installed on your system.
Install the required libraries by running pip install streamlit requests beautifulsoup4 scikit-learn nltk in your terminal.

Clone the Repository:
If you haven't already, clone this repository to your local machine.

Navigate to the Application Folder:
Open your terminal and navigate to the folder containing the application files.
Run the Application:

In your terminal, enter the following command to start the application:
arduino
Copy code
streamlit run IR_Code.py
Use the Application:

Once the application is running, you can access it through your web browser. A local URL will be displayed in the terminal. Open that URL in your browser.
Search for Stock Market Information:

You can enter your stock market-related queries in the provided text input box and submit them.

View Results:
The application will process your query, and the most relevant information will be displayed on the page.

Dependencies
Streamlit
Requests
BeautifulSoup
Scikit-learn
NLTK

Note:
The application uses predefined URLs to fetch information. You can customize the URLs to fetch data from your preferred sources.
